## 📌 Overview
**InstruNet AI** is an AI-powered web application that identifies musical instruments from audio input.  
The system uses machine learning techniques to analyze sound features and predict the instrument being played in real-time.

This project demonstrates the application of **Artificial Intelligence + Audio Signal Processing** in a real-world scenario.

---

## ✨ Features
- 🎧 Upload audio files for instrument detection  
- 🤖 AI-based prediction of musical instruments  
- 📊 Displays prediction results with confidence scores  
- ⚡ Fast and interactive web interface  
- 🌐 Deployed online for real-time access  
- 🔄 Supports multiple instrument classes
